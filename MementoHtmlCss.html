Les balises (titre de niveau 1)

La première des balises pour un fichier html est <!DOCTYPE html> qui indique que c'est un document HTML. C'est une balise orpheline.
La 2ème indique le debut de la partie html : <html> et se referme </html>
La 3ème indique le debut de la partie html : <head> et se referme </head>
La 4ème indique le type d'alphabet utilisé, pour la France : <meta charset="utf-8" />. C'est une balise orpheline.
La 5ème indique le fichier CSS utilisé pour le site : <link rel="stylesheet" href="memento.css" />. C'est une balise orpheline.
La 6ème indique le titre de la page : <title> et se referme </title>. Attention, elle est importante pour le référencement du site.
La 7ème indique le corp de la page html : <body> et se referme </body>

Je signale un morceau de texte (mise en italique par défaut) avec les balises <em> et </em>
Je signale un morceau de texte comme IMPORTANT (mise en GRAS par défaut) avec les balises <strong> et </strong>
Je marque un morceau de texte (surlignage par défaut) avec les balises <mark> et </mark>

les liens avec des ancres

Il faut au préalable rajouter l'attribut id à une balise qui va servir de repère.
On choisit par exemple le titre principal, et on lui attribue l'id Haut :
<h1 id=Haut>Les balises (titre de niveau 1)</h1>
<a href="#Haut">Aller vers l'ancre Haut en début de document</a>
Aller vers l'ancre Haut en début de document

Si l'ancre se situe sur une autre page, il suffit de mettre le nom de la page suivi de l'ancre.
<a href="./page2.html#Haut">Aller vers l'ancre Haut en début du document page 2</a>
exemple : la page 2

les images

L'insertion d'une image se fait avec une balise de type orpheline comme <br />. Attention! L'insertion d'un image se fait impérativement dans un paragraphe <p> </p>.
src : chemin de l'image
alt : texte alternatif si l'image ne peut pas être affichée par le navigateur.
title : infobulle
href : permet l'utilisation d'une miniature pour ne pas pénaliser letemps de chargement de la page.
ce qui donne : <p> Cliquez sur l'image!<br /> <a href="Images/casc.jpg"> <img src="Images/casc10.jpg" alt="Photo de cascade" title="Cliquez pour agrandir" /></a></p>

Cliquez sur l'image!

les figures

Les figures sont des éléments qui viennent enrichir le texte et qui ont du sens :
  • Images
  • Codes sources, mais pas de l'HTML :))
  • Citations
  • Diagrammes
  • ...
Elles peuvent se trouver dans un paragraphe ou en en dehors. Une figure peut comporter plusieurs éléments: images,...
La balise figcaption permet de mettre un titre sous la figure.
ce qui donne :
<p>Connaissez-vous le logiciel Sublime Text ? On peut faire des sites web avec !</p>
<figure>
<img src="Images/Code.png" alt="une capture d'écran" title="Un peu de code" />
<img src="Images/Code2.png" alt="une capture d'écran" title="Un peu de code" />
<figcaption>Un peu de code !</figcaption>
</figure>

Le CSS

Les points clefs

  • CSS = Cascading Style Sheets
  • Permet de mettre en forme la page html
  • Le site https://www.caniuse.com/ permet de vérifier si la technique est supportée par une version du navigateur.
  • On peut écrire le code CSS en 3 endroits :
    1. Dans un fichier (Méthode la plus recommandable)
    2. Au début du fichier HTML
    3. directement dans une balise
    les 2 dernières méthodes ne sont utilisées que pour gérer des exceptions au niveau de la charte graphique du site.
